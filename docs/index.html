Get Started in Minutes

1. Clone and Build

git clone https://github.com/ptescher/roxy.git
cd roxy
cargo build --release

2. Run Roxy

cargo run --release --bin roxy

On first run, Roxy automatically downloads ClickHouse and OpenTelemetry Collector.

3. Trust the CA Certificate

Import ~/.roxy/ca.crt into macOS Keychain and set trust to "Always Trust"

4. Configure Your Apps

# For your backend
HTTP_PROXY=http://localhost:8080 npm run dev

# For your mobile app
# Set iOS Simulator proxy to localhost:8080
